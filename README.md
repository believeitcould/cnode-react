# cnode-react
第一个react练手项目，react框架写的cnode社区，数据来源于[cnode api](http://cnodejs.org/api)，感谢cnode提供的api。

## npm
- redux
- react-redux
- react-route
- redux-thunk
- redux-logger
- antd

## 语法
- es6

## 运行
    npm install
    npm start
